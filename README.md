# Dialog
Dialog 对话框
    AlertDialog:提示对话框
    ProgressDialog:进度对话框
    DataPickerDialog:日期对话框
    TimePickerDialog:时间对话框

    AlertDialog使用：
        AlertDialog继承Dialog
        包含：
            标题：可选
            内容：普通内容 列表等
            按钮：最多支持三个

    AlertDialog中的内容可以包含普通内容，列表等
    普通内容：很容易
    列表：
        单一选择列表
        单一选择列表（带单选按钮）
        多项选择列表（带复选框）
